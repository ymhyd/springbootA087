### 四、项目介绍
项目编号：springbootA087

北工国际健身俱乐部整体框架、功能服务多样化和有效性的高级经验和技术实现方法，旨在完成一个快速、高效、便捷的北工国际健身俱乐部。本系统以会员与管理员两类人，作为目标会员，其中会员主要功能包含会员的注册与登录，查询新闻活动、健身课程、器械信息、健身资讯等，对账号相关信息的修改；管理员主要功能包括了会员、员工、教练、新闻活动、健身课程、课程查询、退课查询等；管理员可以实现最高权限级别的全系统管理
